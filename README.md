# Touch Grass Collective

A modern, responsive website for the Touch Grass Collective movement - encouraging people to get outside, get healthy, and reconnect with real life.

## Features

- 🌱 **Apple-style animations** with smooth parallax effects
- 📱 **Fully responsive** design for all devices
- ♿ **Accessibility-first** with reduced motion support
- ⚡ **High performance** with optimized animations
- 🎨 **Custom design system** with grass, sky, and sunlight themes

## Tech Stack

- **React 18** with TypeScript
- **Vite** for fast development and building
- **Tailwind CSS** for styling
- **Lucide React** for icons
- **Custom animations** with CSS and JavaScript

## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/touch-grass-collective.git
   cd touch-grass-collective
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```

## Design System

### Colors
- **Grass**: Primary green palette (#4CAF50)
- **Sky**: Primary blue palette (#6EC6FF)  
- **Sunlight**: Primary yellow palette (#FFEB3B)

### Typography
- **Inter**: Primary font for headings and body text
- **Caveat**: Decorative font for special elements

### Animations
- Smooth scroll-triggered animations
- Hardware-accelerated transforms
- Reduced motion support for accessibility
